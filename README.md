# SIMS PPOB - Prasetyo

Aplikasi web **SIMS PPOB** (Sistem Informasi Manajemen PPOB) built with React.js + Redux Toolkit sebagai bagian dari technical test di PT Nutech Integrasi.

---

## ✨ Fitur yang Sudah Diimplementasikan

- ✅ **Authentication**
  - Register dengan validasi
  - Login + simpan token
  - Protected Route

- ✅ **User Profile**
  - Menampilkan data profile dari API
  - Update profile (siap dikembangkan)

- ✅ **Dashboard / Home**
  - Menampilkan saldo
  - Daftar layanan (Services)
  - Promo banner

- ✅ **State Management**
  - Redux Toolkit (Auth + Profile)

- ✅ **API Integration**
  - Axios dengan interceptor
  - Error handling
  - Loading state

- ✅ **UI/UX**
  - Responsive design
  - Tailwind CSS
  - Modern & clean interface sesuai mockup

---

## 🛠️ Tech Stack

- **Framework**: React 19 + Vite + TypeScript
- **State Management**: Redux Toolkit
- **Styling**: Tailwind CSS
- **HTTP Client**: Axios
- **Icons**: Lucide React
- **Notification**: React Hot Toast
- **Routing**: React Router DOM v7
